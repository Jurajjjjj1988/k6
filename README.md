# k6 Performance Testing

Performance and load testing setup for backend services using [k6](https://k6.io).  
This project focuses on realistic user load simulation, API response validation, and performance threshold enforcement.

---

## Features
- Ramping VU load model (stages)
- Environment variables for dynamic test data
- Thresholds for response time, error rate and availability
- JSON and HTML results export
- Modular request functions for reusability
